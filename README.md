# Приложение «Заметки»

Одностраничное приложение для создания и управления заметками: поиск, фильтр по тегам, светлая/тёмная тема, сохранение в браузере.

---

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется по адресу, который выведет Vite (обычно `http://localhost:5173`).

Проект доступен на GitHub Pages: [https://pavelgueliev.github.io/Task-A-Notes/](https://pavelgueliev.github.io/Task-A-Notes/)

## Функционал

### Заметки

- **Создание** — кнопка «Добавить» в шапке открывает модальное окно с формой: заголовок, текст, тег, до 3 изображений (макс. 1 МБ каждое).
- **Редактирование** — кнопка «Редактировать» на карточке открывает ту же форму с подставленными данными заметки.
- **Удаление** — кнопка с иконкой корзины на карточке удаляет заметку.
- **Валидация** — при отправке формы обязателен заголовок; при пустом заголовке показывается сообщение «Заголовок обязателен».

### Поиск и фильтры

- **Поиск** — поле в шапке фильтрует заметки по заголовку и тексту (без учёта регистра).
- **Теги** — кнопки «Все», «Личное», «Работа», «Дом» под заголовком страницы; активный тег подсвечивается, список заметок фильтруется по выбранному тегу.
- Поиск и тег работают вместе: сначала применяется поиск, затем фильтр по тегу.

### Светлая / темная тема

- Выбор сохраняется в `localStorage` и при первом заходе учитывается системная тема (`prefers-color-scheme`).

### Хранение данных

- Заметки хранятся в **localStorage** (ключ `notes-storage`) через Zustand persist.
- При изменении схемы данных используется **миграция** (версия хранилища): приведение старых заметок к актуальному формату (заголовок, тег, массив изображений, очистка HTML из текста).

## Стек и инструменты

| Категория        | Технология        | Назначение |
|------------------|-------------------|------------|
| Сборка           | **Vite 7**        | Dev-сервер, HMR, production-сборка |
| UI               | **React 19**      | Компоненты, хуки |
| Стили            | **Tailwind CSS 4**| Утилитарные классы, тёмная тема по классу |
| Состояние        | **Zustand**       | Глобальный стор заметок, поиска и активного тега |
| Сохранение       | **zustand/persist** | Сохранение заметок в localStorage, миграции |
| Линтинг          | **ESLint 9**      | Правила для JS/JSX, React Hooks, React Refresh |
| Форматирование   | **Prettier**      | Единый стиль кода, интеграция с ESLint |
